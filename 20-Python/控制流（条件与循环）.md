# 控制流（条件与循环）

> [!info] 所属：[[Python MOC]] · 上一篇 [[数据类型与数据结构]] · 下一篇 [[函数与作用域]]

## 核心概念

程序默认**自上而下顺序执行**。控制流（control flow）让你能**改变执行路径**：根据条件做不同事（分支）、重复做某事（循环）。这是编程最基础的能力——掌握 `if` 和 `for`/`while`，就能让程序「做决定」和「重复劳动」。

### 顺序执行（Sequential）
默认行为：一行接一行，从上到下执行。**缩进**界定代码块（Python 没有大括号）。

### 条件分支（Conditional）
```python
if 条件:
    ...
elif 另一条件:
    ...
else:
    ...
```
- **真值判断**：`0`、`0.0`、`''`、`[]`、`{}`、`None`、`False` 都是 **falsy**（当假）；其余 truthy。
- **三元表达式**：`值A if 条件 else 值B`（单行条件赋值）。
- **条件组合**：`and`（与）、`or`（或）、`not`（非）。
- **链式比较**：`if 0 < x < 10`（Python 独有的简洁写法）。

### 循环（Loop）
| 语句 | 适用 | 说明 |
| --- | --- | --- |
| `for 变量 in 可迭代对象` | **遍历**已知序列/范围 | 最常用 |
| `while 条件` | 条件为真就**重复** | 不确定次数时用 |
| `range(start, stop, step)` | 生成整数序列 | 配合 for 做计数循环 |
| `enumerate(xs)` | 同时拿**索引和值** | 比手动 i++ 优雅 |
| `zip(a, b)` | **并行**遍历多个序列 | 长度按最短的截 |

### 循环控制
- `break`：**跳出**整个循环。
- `continue`：**跳过**本次，进入下一次。
- `pass`：**空操作占位**（语法上需要一个语句但什么都不做）。
- `else`（循环后的 else）：循环**正常结束**（没被 break）才执行——常用于「搜索找没找到」。

### 推导式（Comprehension）
用一行生成列表/字典/集合，是 Pythonic 的标志：
```python
[x*2 for x in xs if x > 0]            # 列表推导
{k: v for k, v in pairs}               # 字典推导
{x for x in items}                     # 集合推导
(x*2 for x in xs)                      # 生成器表达式（惰性，见 [[装饰器迭代器与生成器]]）
```

### match-case（3.10+）
结构化模式匹配，替代冗长的 if-elif 链：
```python
match command:
    case 'quit':  ...
    case 'go' | 'move':  ...      # 多个模式用 |
    case _ :  ...                  # _ 是通配（类似 else）
```

## 代码示例

```python
# 1) 条件
score = 85
if score >= 90:
    grade = 'A'
elif score >= 60:
    grade = 'B'
else:
    grade = 'C'

# 三元 + 链式比较
status = 'pass' if score >= 60 else 'fail'
if 0 <= score <= 100:           # 链式比较，等价 0<=score and score<=100
    pass

# 2) for 遍历
fruits = ['apple', 'banana', 'cherry']
for f in fruits:
    print(f)

# 带索引（enumerate 比 range(len(...)) 优雅）
for i, f in enumerate(fruits):
    print(i, f)                 # 0 apple / 1 banana / ...

# 并行遍历（zip）
names = ['Ada', 'Bob']; ages = [30, 25]
for name, age in zip(names, ages):
    print(name, age)

# range：计数循环
for i in range(3):              # 0,1,2（不含 3）
    print(i)
for i in range(2, 10, 2):       # 2,4,6,8（start, stop, step）
    print(i)

# 3) while + break/continue
n = 10
while n > 0:
    n -= 1
    if n == 5: break            # 到 5 跳出
    if n % 2 == 0: continue     # 偶数跳过打印
    print(n)                    # 只打印奇数

# 4) 循环的 else（没被 break 才执行——「没找到」语义）
for x in [1, 2, 3]:
    if x == 5:
        print('找到了')
        break
else:
    print('没找到 5')           # 执行这句（循环正常结束）

# 5) 推导式（一行生成）
nums = [1, -2, 3, -4]
doubled_pos = [x * 2 for x in nums if x > 0]   # [2, 6]
word_len = {w: len(w) for w in ['hi', 'world']} # {'hi':2, 'world':5}

# 6) match-case（3.10+）
def handle(cmd):
    match cmd.split():
        case ['go', direction]:  print(f'向 {direction} 走')
        case ['quit']:           print('退出')
        case _:                  print('未知命令')
```

## ⚠️ 易错点 / 面试要点

- **缩进必须一致**：Python 靠缩进分块，混用空格和 Tab 会 `IndentationError`。统一用 **4 个空格**（PEP 8）。
- **`range` 不含上界**：`range(0, 5)` 是 `0,1,2,3,4`，没有 5。新手常见 off-by-one。
- **`while` 当心死循环**：条件最终必须能变假（记得更新循环变量），否则 `while True` 不退出（除非有 `break`）。
- **遍历时修改列表是陷阱**：边 `for` 边 `remove` 元素会跳过元素。要过滤就建新列表（用推导式）或遍历副本 `for x in lst[:]`。
- **判空用真值，别写 `==`**：`if lst:` 比 `if len(lst) > 0:` 更 Pythonic；判 None 用 `if x is None`。
- **`and`/`or` 返回的是操作数不是布尔**：`a or b` 返回第一个 truthy 值（或最后的值），不是 True/False。常用于默认值 `name = input or '匿名'`。
- **`is` vs `==`**：`if x is None` 用 `is`；其它值比较用 `==`。别 `if x == None`。
- **推导式别滥用**：逻辑复杂（嵌套、多条件）时不如普通 for 循环可读。**简单映射/过滤用推导，复杂逻辑用 for**。
- **循环的 `else` 容易误读**：它是「**没 break** 时执行」，不是「条件为假时」。不熟就别用，改用标志变量更清晰。
