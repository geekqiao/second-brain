# 数据类型与数据结构

> [!info] 所属：[[Python MOC]] · 下一篇 [[控制流（条件与循环）]]

## 核心概念

Python 里**一切皆对象**。理解数据类型，关键是抓住两件事：**每个类型的用途**，以及**可变（mutable）vs 不可变（immutable）**——后者是无数 bug 的根源。

### 内置类型一览
| 类别 | 类型 | 示例 | 可变？ |
| --- | --- | --- | --- |
| 数值 | `int`, `float`, `complex`, `bool` | `1`, `3.14` | 不可变 |
| 序列 | `str`, `tuple` | `'hi'`, `(1,2)` | 不可变 |
| 序列 | `list` | `[1,2]` | **可变** |
| 映射 | `dict` | `{'a':1}` | **可变** |
| 集合 | `set` | `{1,2}` | **可变**（`frozenset` 不可变） |
| 空值 | `NoneType` | `None` | 不可变 |

### 可变性（mutable vs immutable）
- **不可变**：值不能被改变。`x = 1; x += 1` 是让 `x` 指向新对象，不是改原对象。
- **可变**：对象本身可被原地修改（list.append、dict[k]=v）。
- **关键陷阱**：函数默认参数用可变对象、嵌套结构的浅拷贝。

### 四大核心容器
- **list**：有序、可变、允许重复。`append/pop/sort/切片`。
- **tuple**：有序、不可变。适合固定结构（坐标、记录）、可做 dict 键。
- **dict**：键值映射，键必须可哈希（不可变）。Python 3.7+ **保持插入顺序**。
- **set**：无序、唯一元素。去重与集合运算（交并差）极快。

### 拷贝：浅 vs 深
- 赋值 `b = a`：只复制引用，指向同一对象。
- 浅拷贝 `a.copy()` / `list(a)` / `a[:]`：复制外层，**嵌套对象仍共享引用**。
- 深拷贝 `copy.deepcopy(a)`：递归复制全部，完全独立。

## 代码示例

```python
# 1) 可变性：int 不可变，list 可变
x = 1
y = x          # y 指向同一个 1
y += 1         # y 指向新对象 2；x 仍是 1
print(x, y)    # 1 2

lst = [1, 2]
lst2 = lst
lst2.append(3) # 原地修改，lst 也变
print(lst)     # [1, 2, 3]（同一对象）

# 2) 四大容器
names = ['Ada', 'Bob']          # list
point = (10, 20)                # tuple，可做 dict 键
ages = {'Ada': 30, 'Bob': 25}   # dict（有序、键唯一）
unique = {1, 1, 2}              # set → {1, 2}

# 3) 常用操作
nums = [3, 1, 2]
nums.sort()                     # 原地排序 [1,2,3]
sl = nums[0:2]                  # 切片 [1,2]
comp = [x * 2 for x in nums if x > 1]  # 列表推导 [2,4,6]

# 4) 浅拷贝 vs 深拷贝
import copy
nested = [[1, 2], [3, 4]]
shallow = nested.copy()         # 浅：外层独立
shallow[0].append(99)
print(nested[0])                # [1, 2, 99] ← 内层仍共享！
deep = copy.deepcopy(nested)    # 深：完全独立

# 5) 优雅遍历
for i, name in enumerate(names):
    print(i, name)              # 0 Ada / 1 Bob
for name, age in ages.items():
    print(name, age)
```

## ⚠️ 易错点 / 面试要点

- **可变默认参数陷阱（经典）**：`def f(x=[]): x.append(1)` —— 默认值在函数定义时只创建一次，反复调用会累积！用 `None` 哨兵：`def f(x=None): x = [] if x is None else x`。
- **浅拷贝不复制嵌套**：`copy()` / 切片只复制一层，嵌套 list 仍共享。
- **`==` vs `is`**：`==` 比较值，`is` 比较身份（是否同一对象）。判 `None` 用 `is None`。
- **小整数缓存**：`a=1; b=1; a is b` 为 True（-5~256 被缓存），别据此误判 `is` 的行为。
- **dict 键必须可哈希**：list 不能做键，tuple 可以（但含 list 的 tuple 也不行）。
- **`sort()` 原地 vs `sorted()` 新建**：`lst.sort()` 改原列表返回 None；`sorted(lst)` 返回新列表。混用易出 `None` 错。
- **字符串不可变**：`s += 'a'` 每次创建新字符串，循环里大量拼接用 `''.join(parts)`。
