# 文本、链接与列表

> [!info] 所属：[[HTML MOC]] · 上一篇 [[文档结构与语义化]] · 下一篇 [[表单与输入]]

## 核心概念

### 标题层级
`<h1>` ~ `<h6>`，数字越小越大越重要。**正确嵌套**：不要从 `h2` 直接跳到 `h4`（跳级影响读屏器导航与 SEO）。

### 段落与强调
- `<p>`：段落。
- `<strong>`：**重要**（默认加粗，但有语义）。
- `<em>`：*语气强调*（默认斜体，有语义）。
- `<b>` / `<i>`：仅视觉加粗/斜体，**无语义**，尽量少用。
- `<br>`：换行（少用，间距交给 [[盒模型|CSS]]）；`<hr>`：主题分隔线。

### 超链接 `<a>`
| 属性 | 作用 |
| --- | --- |
| `href` | 目标地址（绝对/相对/锚点 `#id`） |
| `target="_blank"` | 新标签页打开 |
| `rel="noopener noreferrer"` | 新标签页打开时的**安全必备** |
| `download` | 提示浏览器下载而非导航 |

### 列表
- `<ul>` + `<li>`：无序列表（项目符号）。
- `<ol>` + `<li>`：有序列表（数字），`start` / `reversed` / `value` 可控制。
- `<dl>` + `<dt>`（术语）+ `<dd>`（描述）：定义列表，适合键值对/术语表。

### 字符实体
保留字符要用实体：`&lt;`(`<`)、`&gt;`(`>`)、`&amp;`(`&`)、`&nbsp;`（不换行空格）。

## 代码示例

```html
<!-- 标题与强调 -->
<h2>购物清单</h2>
<p>请于<strong>周五前</strong>完成采购，<em>务必</em>带上环保袋。</p>

<!-- 链接：安全的新标签页打开 -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  外部链接（新标签页）
</a>

<!-- 页内锚点跳转 -->
<a href="#section-2">跳到第二节</a>
<!-- ... -->
<h2 id="section-2">第二节</h2>

<!-- 下载链接 -->
<a href="/files/report.pdf" download>下载报告</a>

<!-- 无序列表 -->
<ul>
  <li>苹果</li>
  <li>香蕉</li>
</ul>

<!-- 有序列表（倒序） -->
<ol reversed>
  <li>第三步</li>
  <li>第二步</li>
  <li>第一步</li>
</ol>

<!-- 定义列表 -->
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
</dl>
```

## ⚠️ 易错点 / 面试要点

- **`target="_blank"` 必须配 `rel="noopener noreferrer"`**：否则新页面能通过 `window.opener` 操纵原页面（钓鱼风险）。现代浏览器默认 noopener，但写上更保险。
- **不要用 `<a>` 当按钮**：`<a>` 用于「跳转/导航」，`<button>` 用于「触发动作」。混用会破坏键盘与读屏体验。
- **`<br>` 滥用制造间距**：布局间距应交给 CSS（`margin`），`<br>` 只用于内容本身需要换行（如诗歌、地址）。
- **列表只能直接包含 `<li>`**：`<ul>` 的直接子元素必须是 `<li>`，不能直接放文本或 `<div>`。
- **空 `<a href="#">`**：`#` 会跳到顶部并写入历史。占位链接用 `<button>` 或 `href="javascript:void(0)"` 都不好——尽量用真实路由或 `<button>`。
