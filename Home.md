## 🧭 如何使用这个知识库

1. **想建立全局观** → 从 [[🗺️ 知识地图 MOC]] 出发，按广度笔记浏览一遍。
2. **想深入某个重点** → 进入对应科目的 **MOC**，按「学习路径」顺序逐篇精读。
3. **复习/查漏** → 每篇深度笔记末尾有 `⚠️ 易错点 / 面试要点`，适合快速回顾。
4. **遇到相关概念** → 顺着笔记里的 `[[双链]]` 跳转，知识会自动连成网。

> [!note] 写作约定
> - 语言：中文为主，技术术语保留英文（如 closure、flexbox、promise）。
> - 所有笔记带统一 frontmatter（`tags` / `aliases` / `created` / `status`）。
> - 笔记顶部有 `> [!info]` 导航条，链接回所属 MOC 与上下篇。
